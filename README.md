# Marinara Agents

Official downloadable agents and capability packages for [Marinara Engine](https://github.com/Pasta-Devs/Marinara-Engine).

This repository is the source and release catalog for optional first-party packages. Marinara Engine installs published, checksum-verified artifacts into its data directory. Package source does not belong in the base Engine distribution.

The package API and catalog are under active development in [Marinara Engine issue #3612](https://github.com/Pasta-Devs/Marinara-Engine/issues/3612).

## Planned packages

- Existing Marinara agents
- Hierarchical spatial maps for Roleplay and Game
- Conversation audio and video calls
- UNO, Chess, Poker, 8-Ball Pool, Tic-Tac-Toe, and Rock-Paper-Scissors

Only repository releases referenced by the signed official catalog are installable as trusted executable packages.
